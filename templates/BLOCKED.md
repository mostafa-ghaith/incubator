# BLOCKED.md

> Open questions that require Owner input. Orchestrator writes here. Owner answers here or in OWNER_INBOX.md.

If this file has no entries below, the workflow is not blocked.

Format:

```
## BLOCKED-NNN — <one-line title>
- raised_by: <agent>
- raised_at: <ISO timestamp>
- condition: <ESCALATION.md P1.N rule>
- context: <what we were doing>
- question: <the actual question>
- options:
    - A) <option> — tradeoff: <...>
    - B) <option> — tradeoff: <...>
    - C) <option> — tradeoff: <...>
- recommendation: <A/B/C and why>
- impact_if_no_answer: <what can't proceed>
- can_proceed_on: <list of tickets/work that IS proceeding>
- resolution: <Owner fills this>
- resolved_at: <ISO timestamp when Owner answers>
```
